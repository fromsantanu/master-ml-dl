
# **Lesson 12 — Visualization Basics (matplotlib, seaborn)**

*Statistical Visualization for Data Analysis*

Visualization is an essential tool in data science for understanding distributions, relationships, and anomalies in data.
Python provides two primary libraries for this:

* **matplotlib** — low-level plotting library
* **seaborn** — high-level statistical visualization library built on top of matplotlib

This lesson explains both libraries, core plot types, and when to use each.

---

# **1. Matplotlib: Low-Level Plotting Library**

Matplotlib provides fine-grained control over the rendering of plots.

### **Key Characteristics**

* Procedural or object-oriented interface
* Full customizability (axes, ticks, markers, colors)
* Useful for building detailed, publication-quality figures

### **Basic Structure**

Matplotlib uses a figure–axes model:

```
Figure → entire canvas
Axes → single plot area
Axis → x/y coordinate lines
```

### **Common Commands**

```python
import matplotlib.pyplot as plt

plt.plot(x, y)
plt.scatter(x, y)
plt.bar(categories, values)
plt.hist(x, bins=20)
plt.show()
```

---

# **2. Seaborn: Statistical Visualization Library**

Seaborn abstracts many low-level operations and produces visually appealing plots with minimal code.

### **Key Characteristics**

* Built on top of matplotlib
* Automatically handles themes, grids, color palettes
* Specialized for statistical visualization
* Integrates well with pandas DataFrames

### **Typical Usage**

```python
import seaborn as sns

sns.histplot(data=df, x="age")
sns.boxplot(data=df, y="salary")
sns.scatterplot(data=df, x="height", y="weight")
```

---

# **3. Core Plot Types and Their Purpose**

Visualization types are chosen based on the structure of the data:

---

## **A. Univariate Numerical Data**

### **1. Histogram**

Purpose: visualize distribution, skewness, modality
Mathematical basis: approximates PDF by binning

[
\text{count in bin}_k = \sum_i \mathbf{1}(x_i \in \text{bin}_k)
]

### **2. Kernel Density Estimate (KDE)**

Smooth estimate of PDF:

[
\hat{f}(x) = \frac{1}{nh} \sum_{i=1}^n K\left(\frac{x - x_i}{h}\right)
]

Seaborn: `sns.kdeplot()`

---

## **B. Univariate Categorical Data**

### **1. Bar Plot**

Shows frequency of each category:

[
\text{freq}(c_j) = \sum_i \mathbf{1}(x_i = c_j)
]

Useful for understanding class imbalance.

---

# **C. Bivariate Numerical Data**

### **1. Scatter Plot**

Visualizes joint distribution and linear (or nonlinear) relationships.

* Pattern → correlation structure
* Clusters → potential segmentation
* Outliers → anomalies

### **2. Line Plot**

For time-series or sequential data.

[
x_t \text{ vs } t
]

---

# **D. Numerical vs Categorical**

### **1. Box Plot**

Shows median, quartiles, and outliers:

* Q1, Q3
* Median
* IQR = Q3 − Q1
* Outliers: points beyond 1.5 × IQR

Seaborn: `sns.boxplot()`

### **2. Violin Plot**

Combines KDE + boxplot.

Useful for comparing distributions between groups.

---

# **E. Correlation and Multivariate Analysis**

### **1. Heatmap**

Correlation matrix:

[
R_{ij} = \rho(X_i, X_j)
]

Visualizes:

* Strong positive correlation
* Strong negative correlation
* Redundancy between features

Seaborn: `sns.heatmap(df.corr())`

### **2. Pair Plot (Scatterplot Matrix)**

Shows pairwise scatter plots and histograms of all features.

Useful for initial multivariate exploration.

# **4. Example Visualization Workflow (House Price Dataset)**

Given features:

* Size (numerical)
* Bedrooms (numerical)
* City (categorical)
* Price (numerical, target)

### Step 1 — Distribution Plots

* Histogram of Price
* KDE plot of Size

### Step 2 — Numerical Relationship

* Scatterplot(Size vs Price)
  → check linearity for regression

### Step 3 — Categorical Impact

* Boxplot(Price by City)
  → location premium detection

### Step 4 — Correlation Heatmap

To detect multicollinearity:

* Size ↔ Bedrooms
* Price ↔ Size

### Step 5 — Outlier Visualization

* Boxplot of Price
  → detect luxury houses as outliers

# **5. Matplotlib vs Seaborn — When to Use Which**

| Task                      | Best Library | Reason                        |
| ------------------------- | ------------ | ----------------------------- |
| Custom, low-level control | matplotlib   | Full flexibility              |
| Quick statistical plots   | seaborn      | Automatic styling             |
| Heatmaps, KDE, boxplots   | seaborn      | Built-in statistical features |
| Publication-grade figures | matplotlib   | Precision control             |
| Working with DataFrames   | seaborn      | Column-name support           |

Seaborn simplifies 80% of EDA tasks; matplotlib remains essential for customization.

---

# **6. Practical Guidelines**

1. Always plot distribution of every feature before modeling.
2. Use scatterplots to detect nonlinear relationships.
3. Verify correlation structure with heatmaps.
4. Use boxplots for outlier analysis.
5. For large datasets, sample data for speed.
6. Apply log-transforms for right-skewed distributions before visualization.

---

# **Summary**

* Matplotlib provides low-level plotting APIs with full control.
* Seaborn provides high-level statistical plots with minimal code.
* Use histograms, KDE, boxplots, and scatterplots for univariate/bivariate analysis.
* Use heatmaps and pairplots for multivariate exploration.
* Visualization is an essential part of understanding data structure and modeling strategy.
